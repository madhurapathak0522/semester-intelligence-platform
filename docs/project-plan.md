# Semester Intelligence Platform (SIP)

## Project Goal

Build an academic planning platform that helps engineering students reduce exam stress, track subjects, manage topics, and plan revisions.

---

## Phase 1 (MVP)

### Features

* Semester selection
* Subject dashboard
* Topic progress tracking
* Weak-topic detection
* Study planner
* Revision reminders

---

## User Flow

Student opens app
↓
Select semester
↓
View subjects
↓
Track completed topics
↓
Weak topics identified
↓
Study plan generated
↓
Revision reminders shown
